# Exploratory Analysis of COVID-19 Data

## Project Overview
This project focuses on exploring data about the COVID-19 pandemic worldwide, including infection rates, mortality rates, and vaccination progress.

## Data Sources
* **CovidDeaths.csv**: Data on the number of infections and deaths.

* **CovidVaccinations.csv**: Data on vaccinations and testing.

## Skills & Tools Used
* **SQL**: Joins, CTEs, Temp Tables, Window Functions, Aggregate Functions, Creating Views.

* **Database**: SQL Server.

* **Visualization**: Tableau

## How to use
1. Import two CSV files into SQL Server.

2. Run the `.sql` files to perform the analysis.
